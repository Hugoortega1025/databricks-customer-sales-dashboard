# Databricks Customer and Sales Dashboard Analysis

## Overview
This project analyzer customer and product sales data using SQL in Databricks and presents key business insights through dashboards. The goal of the analysis was to explore sales performance, customer distribution, and product-level sales trends using metrics and visualizations. 

Two dashboards were devleoped using SQL queries to summarize and provide key metrics, visuals, and trends of the given dataset.

## Sales Performance Dashboard
The sales performance dashboard provides an essential overview of the key sale metrics and product performance.
- Total Sales: $29.36M
- Total Quantity Sold: 60.42K units
- Total Profit: $11.69M
- Monthly Sales trend
- Distribution of total sales across product categories

## Customer Insight Dashboard
The customer dashboard reports on customer related data and behaviors.
- Customer distirbution across age
- Total Sales by Customer


## SQL
Both dashboards were developed thorugh SQL queries written in Databricks to explore the dataset, involving querying:
- Total sales
- analyzing sales trend over time
- counting customers and products
- exploring customer distributions

### Data Tables
- fact_sales
- dim_products
- dim_customers
